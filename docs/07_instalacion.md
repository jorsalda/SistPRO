# Instalación

## Requisitos
- Python 3.10+
- PostgreSQL

## Pasos

1. Clonar repositorio
git clone ...

2. Crear entorno virtual
python -m venv .venv

3. Activar entorno
source .venv/bin/activate

4. Instalar dependencias
pip install -r requirements.txt

5. Configurar variables de entorno (.env)

6. Migraciones
flask db upgrade

7. Ejecutar
python run.py
