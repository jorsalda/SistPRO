# SistPROF

## Descripción
SistPROF es una plataforma web para la gestión educativa de instituciones, diseñada para centralizar procesos académicos, administrativos y de evaluación.

## Objetivo
Optimizar la administración escolar mediante herramientas digitales que faciliten el seguimiento académico, control de usuarios y toma de decisiones.

## Características principales
- Plataforma web multiusuario
- Gestión por roles (admin, docente, etc.)
- Control de permisos
- Escalable y modular
- Integración futura con evaluaciones tipo ICFES

## Tipo de sistema
- SaaS educativo (Software como servicio)
- Arquitectura basada en Flask + PostgreSQL
